DeASCIIfy Turkish Letters

Converts some un-accented (e.g. due to a non-Turkish keyboard layout) Turkish letters to their basic Latin counterparts (C -> Ç, G -> Ğ, i -> ı, I -> İ, O -> Ö, S -> Ş, and U -> Ü) based on certain rules. Uses f/deasciifier, which is based on meacer/deasciifier, which is based on the original work of denizyuret/deasciify.
